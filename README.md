## 🔹 Features

- Upload Excel (.xlsx / .xls) and convert to PDF instantly  
- Download PDF directly in browser  
- Top & middle ad placeholders included  
- Responsive and clean UI  
- Works fully client-side (no server required)  

---

## 🔹 How to Use

1. Click **Upload Excel** and select your file  
2. Click **Convert to PDF**  
3. Download your PDF using the **Download** button
